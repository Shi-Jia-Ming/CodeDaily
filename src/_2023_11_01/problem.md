## 有效括号序列

#### 描述

给出一个仅包含字符`(`, `)`，`{`, `}`，`[`, `]`的字符串，按段给出的字符串是否是合法的括号序列。

括号必须以正确的顺序关闭，`()`和`()[]{}`都是合法的括号序列。但是`(]`不合法。

数据范围：字符串长度`0 <= n <= 10000`。
要求：空间复杂度`O(n)`，时间复杂度`O(n)`。

#### 示例

```txt
输入：'['
返回值：false
```

```txt
输入：'[]'
返回值：true
```
